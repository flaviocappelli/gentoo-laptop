# Gentoo Laptop Installation
This repository contains the configuration of my Gentoo laptop (Lenovo Legion
Pro 5 16ADR10: CPU Ryzen 8940HX, iGPU Radeon 610M, dGPU RTX 5070, multilib,
Plasma 6, systemd). Since most settings are shared with my Gentoo desktop (see
the "gentoo-desktop" repository), this repo <ins>contains only the files that
differ from the desktop configuration</ins>.
