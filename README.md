# Gentoo Laptop Installation
This repository contains the Gentoo configuration for my laptop Lenovo Legion Pro 5 16ADR10 (CPU Ryzen 8940HX, iGPU Radeon 610, dGPU NVidia RTX 5070, multilib, PLASMA 6, systemd). Most configuration files (and scripts) are shared with the desktop (see my "gentoo-desktop" repo), so this repository only contains files that differ from the desktop configuration.
